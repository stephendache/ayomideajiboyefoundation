# Ayomide Ajiboye Foundation Website
Static, mobile-first website for the Ayomide Ajiboye Foundation (AAF).

## Stack

---
## Run

```bash
git clone https://github.com/stephendache/ayomideajiboyefoundation.git
cd ayomideajiboyefoundation
open index.html
```
